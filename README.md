# -Goblin-Empire-Mine-City-
 Гра повністю адаптована для смартфонів.  ## 🚀 Як грати: 1. Відкрийте посилання на гру. 2. Натисніть "Наняти гобліна". 3. Перетягніть гобліна на руду, щоб почати видобуток. 4. Схрещуйте однакових гоблінів, щоб подвоїти їхній урон. 5. Перейдіть у вкладку "Місто", щоб пришвидшити розвиток.
